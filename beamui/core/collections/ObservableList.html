    Class ObservableList

    Array with ability to track its changes via beforeChange and afterChange signals.

    class ObservableList(T)
      
    if (is(T == struct) || isScalarType!T || isDynamicArray!T);

    remove (index) Remove one item at index. The index must be in range
    removeAll () Remove the whole content

    replace (index, item) Replace one item at index. The index must be in range
    replaceAll (array) Replace the whole content
    replaceItems (index, count, array) Replace count items at index with items from array. The indices must be in range

    Example

    import beamui.core.collections;
import beamui.core.logger;

auto list = new ObservableList!int;
list.afterChange ~= (ListChange change, uint index, uint count) {
    Log.fd("%s at %s with %s items", change, index, count);
};

list.append(1);
list.insertItems(0, [1, 2, 3]);
